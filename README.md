## Architecture

```
vidyalaxmi/
├── apps/flutter_app/        ← Flutter (Mobile + Web, 60fps)
├── services/gateway/        ← Node.js API Gateway (Express + TypeScript)
├── services/ai-engine/      ← FastAPI AI Microservice (ADL, Bhashini, Scroll PDF)
├── supabase/migrations/     ← PostgreSQL schema + RLS policies
├── DESIGN.md                ← Design system specification (source of truth)
└── docker-compose.yml       ← Local development stack
```

## Quick Start

### Prerequisites

- Flutter SDK 3.22+ (`brew install --cask flutter`)
- Node.js 22+ (already installed)
- Python 3.12+
- Docker Desktop

### 1. Environment

```bash
cp .env.example .env
# Fill in SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_SERVICE_ROLE_KEY
```

### 2. Database

```bash
# Create project at supabase.com, then run migrations:
supabase db push  # or paste migrations/*.sql into Supabase SQL editor
psql $SUPABASE_URL -f supabase/seed.sql
```

### 3. Backend Services

```bash
docker compose up --build
# Gateway: http://localhost:3000/health
# AI Engine: http://localhost:8000/health
# AI Engine docs: http://localhost:8000/docs
```

### 4. Flutter App

```bash
cd apps/flutter_app
flutter pub get
flutter run \
  --dart-define=SUPABASE_URL=your-url \
  --dart-define=SUPABASE_ANON_KEY=your-key \
  --dart-define=GATEWAY_URL=http://localhost:3000 \
  --dart-define=AI_ENGINE_URL=http://localhost:8000
```

### 5. Install Flutter (macOS)

```bash
# Fix Homebrew permissions first:
sudo chown -R $(whoami) /opt/homebrew
brew install --cask flutter
flutter doctor
```

## Game Mechanics

### Learning Loop

```
Skill Tree → Jingle Hook (30-60s song) → Boss Fight (ADL-powered quiz) → Reward (L-Coins) → Manga Scroll PDF
```

### ADL Algorithm

```
weighted_score = 0.6 × accuracy + 0.4 × (1 − response_time / 15s)
  > 0.7 → difficulty +1 (max 5)
  < 0.3 → difficulty -1 (min 1)
  else  → maintain
```

### L-Coin Economy

```
earned = base_reward × accuracy_multiplier × streak_bonus
```
